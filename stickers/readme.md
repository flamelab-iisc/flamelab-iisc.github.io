These files can be printed in sticker format from printo shop. <br>
They are meant to be used in lab for sticking on computers and furniture <br>
to identify FLAME lab property. <br> <br>

`flame_lab_stickers.pdf` contains 16 x 4 = 64 plain text boxes written as FLAME LAB. <br>
`flame_logo_stickers.pdf` contains 14 x 5 = 84 FLAME lab logos. <br><br>

Source file of `flame_lab_stickers.pdf` is `flame_lab_stickers.xlsx`. <br>
Source file of `flame_logo_stickers.pdf` is `flame_logo_stickers.docx`. <br>
